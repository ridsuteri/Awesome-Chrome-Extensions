How to use these extensions?

If you'd like to use these extensions for yourself, or perhaps use parts of it and modify the rest, here's how you will need to set it up:

  1. Clone this repository.
  2. Find out the extension that excites you.
  3. If you'd like to further customize it or make any other edits to the page, you will likely want to do that in either the popup.html file or the script.js file.
  4. Once you've made your desired changes (if any), open Chrome and navigate to chrome://extensions/.
  5. Click the toggle in the upper-right corner marker "Developer Mode".
  6. You should then see three new buttons towards the top of your screen.
  7. Click "Load Unpacked", and then select the folder containing this repository.
  8. You can then turn off Developer Mode.
  9. Now, when you open up a new tab, you should see the customized page!
