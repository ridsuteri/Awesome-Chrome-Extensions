## Time Zone Finder 

This is a simple extension that displayes the current timezone at any location. Also added digital clock here.

<h3>Tech Stack:</h3>
<ul>
  <li>HTML5</li>
  <li>CSS</li>
  <li>Javascript</li>
</ul>
